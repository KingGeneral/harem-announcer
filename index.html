Installation guide

1. Download the sound files.

2. Download your custom installation script (install.bat).

3. Extract the ZIP file.

4. Place install.bat into the extracted harem_spring_2016 folder.

5. Run install.bat and follow the instructions.

6. (Optional) If you have the Rise announcer, equip the GlaDOS announcer for additional lines.
